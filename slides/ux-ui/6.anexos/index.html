1. Integración de APIs REST con Node.js

Crear endpoints básicos usando Express:

            
  import express from 'express';
  const app = express();
  app.use(express.json());

  // GET básico
  app.get('/usuarios', (req, res) => {
    res.json([{ id: 1, name: 'Ana' }]);
  });

  // POST con JSON
  app.post('/usuarios', (req, res) => {
    const { name, email } = req.body;
    res.status(201).json({ id: Date.now(), name, email });
  });

  app.listen(3000, () => console.log('API corriendo en http://localhost:3000'));

3. Supabase como backend de servicios

Inicializa el cliente y realiza operaciones:

npm install @supabase/supabase-js

import { createClient } from '@supabase/supabase-js';
const supabase = createClient(SUPABASE_URL, SUPABASE_ANON_KEY);

// Leer registros
const { data, error } = await supabase.from('posts').select('*');

// Insertar registro
await supabase.from('posts').insert([{ title, content }]);
