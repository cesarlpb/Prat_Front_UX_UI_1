Introducción a React Native

React Native es un framework de JavaScript para construir aplicaciones móviles nativas en iOS y Android usando React.

  • Creado por Facebook (Meta) en 2015.
  • Permite compartir gran parte del código entre plataformas.
  • Renderiza componentes nativos, no WebViews.
  • Desarrollo con JSX y JavaScript moderno.

Ejemplos de componentes

Componente funcional:

import React from 'react';
import { View, Text, Button } from 'react-native';

function Saludo({ nombre }) {
  return (
    <View>
      <Text>¡Hola, {nombre}!</Text>
      <Button title="Presióname" onPress={() => alert('¡Hola!')} />
    </View>
  );
}

export default Saludo;

Componente de clase:

import React, { Component } from 'react';
import { View, Text, TouchableOpacity } from 'react-native';

class Contador extends Component {
  state = { count: 0 };

  render() {
    return (
      <View>
        <Text>Clics: {this.state.count}</Text>
        <TouchableOpacity onPress={() => this.setState({ count: this.state.count + 1 })}>
          <Text>Incrementar</Text>
        </TouchableOpacity>
      </View>
    );
  }
}

export default Contador;

¿Qué ofrece React Native?

  • Componentes nativos: View, Text, Image, etc.
  • Estilos con Flexbox y StyleSheet.create.
  • Hot Reload y Fast Refresh para iteración rápida.
  • Acceso a APIs nativas: Cámara, GPS, Sensores.
  • Integración con librerías de la comunidad (React Navigation, Redux, etc.).
  • Expo CLI para arranque rápido y despliegue simplificado.

Componentes básicos

  • View – contenedor genérico
  • Text – mostrar texto
  • Image – imágenes locales o remotas
  • ScrollView – contenedor desplazable
  • FlatList – listas optimizadas
  • TextInput – entrada de texto

Estilos en React Native

  • Definidos en JavaScript con objetos.
  • StyleSheet.create({ … }) para mejores validaciones.
  • Flexbox por defecto para diseño responsivo.
  • No existen clases CSS; todo es inline o en objetos StyleSheet.

Navegación

  • React Navigation – librería más popular.
  • Stack Navigator, Tab Navigator, Drawer Navigator.
  • Configuración con componentización y hooks (useNavigation).

Acceso a APIs nativas

  • PermissionsAndroid para permisos en Android.
  • Geolocation, Camera, Accelerometer, etc.
  • Uso de módulos nativos nativos o bridged modules.

Herramientas y entorno

  • Expo CLI: iniciar proyectos sin configuración nativa.
  • React Native CLI: control total sobre código nativo.
  • Debugging con React DevTools y Flipper.
  • Testing con Jest y React Native Testing Library.
